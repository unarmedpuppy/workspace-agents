# Agents Directory

AI workflow documentation for workflow-agents.

## Structure

```
agents/
├── plans/        # Implementation plans (commit multi-session work)
│   └── local/    # Scratch work (gitignored)
├── reference/    # Project-specific documentation
├── personas/     # Agent personalities for domain expertise
└── skills/       # Executable skills (Anthropic standard)
```

## Quick Reference

| Directory | Purpose | When to Use |
|-----------|---------|-------------|
| **plans/** | Track multi-step work | Starting features, complex changes |
| **plans/local/** | Ephemeral scratch | Exploration, single-session notes |
| **reference/** | Deep documentation | API patterns, architecture decisions |
| **personas/** | Domain expertise | Recurring specialized workflows |
| **skills/** | Executable actions | Repeatable, validated processes |

## Creating Content

### Plans

Create a markdown file in `plans/` with:
- Clear objective
- Phased approach with tasks
- Success criteria

### Reference Docs

Create topic-focused docs in `reference/`:
```markdown
# Topic Name

Overview of the topic.

## Key Concepts

## Examples

## Common Issues
```

### Personas

Create domain experts in `personas/`:
```markdown
# Domain Agent

Brief description of expertise.

## Capabilities

## Key Files

## When to Use
```

### Skills

Use `/skill skill-creator` to create Anthropic-compliant skills.
